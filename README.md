# Voitto Kasino
Full Stack -kurssin projektityönä Reactilla koodattu nettikasino.

## Kokeile appia
1. Kloonaa repo:
```git clone https://github.com/januzgi/React-Voitto-Kasino.git```

1. Mene kansioon ```voitto-kasino```

3. Asenna appi:
```npm install ```

4. Aja appi:
```npm start ```
